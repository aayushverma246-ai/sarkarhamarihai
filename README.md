# SarkarHamariHai

AI-powered tool for finding and applying to government exams and jobs.

## Features

- AI Recommendation System (Gemini powered)
- Real-time notifications for exam updates
- Interactive dashboard for managing applications
- Support for multiple languages (Hindi, Marathi, English)

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express, Turso (SQLite)
- **AI**: Google Gemini API
- **Tooling**: TypeScript, Lucide React

## Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Seed the database:
   ```bash
   npm run seed
   ```
